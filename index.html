App Tindev

Tindev is an application that seeks to facilitate social contact between software developers. Using Github API to fetch information shown on screen such as photo, name and bio. The other user can accept or reject that dev, as it does in the referral application. If there are two likes between the same developers, a match animation for both of them also happens. So they can perform contact and perform the tasks they were looking for in the technical qualities of the developer. I used Node.Js to create the application backend, React for the web front end and React-Native in the mobile version. For real time communication I used the Socket.IO technology, using the socket.io and client socket.io libraries.
